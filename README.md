An e-commerce web app made with **React, Redux, ChakraUI** for the frontend and **Node, Express, MongoDB** for the backend. It uses **Stripe** for payments.

  
## To Start: 
  
  1. Clone this repo.
  
  2. `npm install`

  3. `npm seed`
  
  3. `cd client && npm install`
  
  4. `cd .. && cd server && npm install`

  5. `cd .. && npm dev`
